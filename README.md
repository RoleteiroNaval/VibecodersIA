# VibeCoders IA - Game OS & Hub do Projeto

Hub interativo do VibeCoders IA, criado para organizar lives, projetos, missoes, comunidade e progresso publico do canal em uma estetica Cavepunk Dev / Neon Primal Coding.

## Conceito

VibeCoders IA e um canal/comunidade de criacao ao vivo onde projetos reais sao construidos com IA, codigo, design, automacoes, videos, sites, apps e experimentos criativos.

- Operadores: Victor & Zeva
- Guardiao de seguranca: Woll chard
- Identidade visual: Cavepunk Dev / Neon Primal Coding
- Slogan: IA - Codigo - Vibe
- Comunidade: O Cla de Madrugada

## Recursos do Game OS

- Arvore de missoes interativa com 20 etapas de evolucao.
- Terminal funcional com comandos `help`, `status`, `clear` e codigos secretos.
- Sistema de XP, nivel, dependencias e desbloqueio progressivo.
- Modo OBS Live Overlay para transmissoes.
- Simulador de chat do Cla ao vivo.
- Arsenal IA com prompts, projetos e stack do canal.

## Como Rodar

Abra o arquivo `index.html` diretamente no navegador ou rode um servidor local:

```bash
python3 -m http.server 4174
```

Depois acesse:

```text
http://localhost:4173/
```
